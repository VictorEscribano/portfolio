# Víctor Escribano García — Portfolio

Personal portfolio website. **Live:** https://victorescribano.github.io/portfolio/

AI Engineer · Computer Vision · Robotics Perception · Deep Learning.

## Stack
Static site — plain HTML, CSS and vanilla JavaScript. No build step, no dependencies.

```
.
├── index.html          # main one-page site
├── styles.css          # global styles
├── script.js           # nav, scroll-reveal, typing effect
├── project.css         # project case-study pages
├── projects/           # one detail page per project
└── assets/
    ├── Victor_Escribano_CV.pdf
    └── img/            # images, gifs and project videos
```

## Run locally
```bash
python3 -m http.server 8000
# open http://localhost:8000
```
